# 🎓 Student Grade Calculator (CLI-Based)

## 📌 Project Description

This is a simple Command Line Interface (CLI) based Student Grade Calculator built using Python. The program calculates total marks, percentage, and grade based on marks entered by the user.

This project is beginner-friendly and demonstrates basic Python programming concepts.

## 🚀 Features

* Input number of subjects
* Enter marks for each subject
* Calculate total marks
* Calculate percentage
* Automatic grade calculation
* Display remarks

## 🛠️ Technologies Used

* Python 3
* CLI (Command Line Interface)
* Visual Studio Code (VS Code)

## 📈 Grading System

| Percentage | Grade | Remarks      |
| ---------- | ----- | ------------ |

| 60 - 69    | B     | Good         |
| 50 - 59    | C+    | Satisfactory |
| 40 - 49    | C     | Acceptable   |
| 35 - 39    | D     | Basic        |
