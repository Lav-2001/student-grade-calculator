# Student Grade Calculator Documentation

## 1. Project Title

Student Grade Calculator (CLI-Based)

## 2. Project Description

The Student Grade Calculator is a Command Line Interface (CLI) based Python application that calculates the total marks, percentage, and grade of a student based on the marks entered by the user.

This project helps beginners understand:

* Functions
* Loops
* Conditional Statements
* Lists
* User Input Handling

## 3. Features

* Input number of subjects
* Enter marks for each subject
* Calculate total marks
* Calculate percentage
* Assign grade automatically
* Display remarks

## 4. Technologies Used

* Python 3
* Command Line Interface (CLI)

## 5. How the Program Works

Step 1: User enters number of subjects
Step 2: User enters marks for each subject
Step 3: Program stores marks in a list
Step 4: Program calculates total marks
Step 5: Program calculates percentage
Step 6: Program assigns grade
Step 7: Program displays result

## 6. Grading System

| Percentage | Grade | Remarks      |
| ---------- | ----- | ------------ |
| 90 - 100   | A+    | Outstanding  |
| 80 - 89    | A     | Excellent    |
| 70 - 79    | B+    | Very Good    |
| 60 - 69    | B     | Good         |
| 50 - 59    | C+    | Satisfactory |
| 40 - 49    | C     | Acceptable   |
| 35 - 39    | D     | Basic        |
| Below 35   | NG    | Not Graded   |

## 7. Function Description

### grade_calculator(mark)

This function calculates the total marks, percentage, and grade.

Parameters:

* mark (int): Number of subjects

Returns:

* None
